# Diffusers — 总览

Diffusers 是 Hugging Face 的扩散模型库，支持 Stable Diffusion、SDXL、ControlNet、LoRA、Text2Image、Image2Image 等主流生成式模型。

用途：

* **推理（Inference）**
* **微调（LoRA Training）**
* **管线组合（Pipeline）**
* **模型组件化拼装**
* **多调度器（Scheduler）选择**

适用于：

* 文生图
* 图生图
* 高清修复
* 控制条件生成（如 ControlNet）
* LoRA 训练/加载/融合

本 AI Doc 提供一个针对 agentic coding 的结构化知识库，使模型能快速理解并正确调用 Diffusers API。
